Arduino UNO pc watercooling monitor.

parts list:

Arduino Uno R3
Adafruit 2.2" 18-bit color TFT LCD display with microSD card breakout
6 10k resistors
4 XSPC G1/4" Plug with 10k Sensor
1 type 2 10k NTC thermistor (for ambient)
Koolance INF-FM19 coolant flow meter

system is connected in this order.

--> PUMP --> RAD IN TEMP --> RAD IN --> RAD OUT --> RAD OUT TEMP --> CPU IN -->> CPU OUT -->> GPU IN TEMP --> GPU IN --> GPU OUT --> GPU OUT TEMP --> FLOW --> PUMP

Watercooling Loop
EK D5 pump / res combo.
Koolance 140mm double radiator.
EK ASUS Maximus Hero VIII motherboard block.
EK GTX Titan waterblock.
